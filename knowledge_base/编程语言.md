# pytorch
在pytorh的反向传播中,上层传入到本层的内容的是最终输出L对上层输入y(即本层输出)的一个求导矩阵(通过马尔可夫链层层计算)。\
本层会计算本层的雅可比矩阵(本层输出y对输入x的求导)，然后与上层传入进行计算得到最终输出对本层输入的求导矩阵，再传递给下一层。pytorch并不会显式地计算雅可比矩阵，而是通过使用矩阵乘法 `y = x @ W` 的反向：`grad_x = grad_output @ W.T`，`grad_W = x.T @ grad_output`来得到梯度\
.grad保存的是梯度，即L对x的偏导，新的梯度与旧的梯度是累加的。\
当使用detach将计算移出计算图后，梯度不会累加这次计算地梯度\
